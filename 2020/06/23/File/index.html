File

前言

构造函数

Constructor and Description 备注
File(File parent, String child) 从父抽象路径名和子路径名字符串创建新的 File实例。 父抽象名：文件夹；
File(String pathname) 通过将给定的路径名字符串转换为抽象路径名来创建新的 File实例。
File(String parent, String child) 从父路径名字符串和子路径名字符串创建新的 File实例。
File(URI uri) 通过将给定的 file: URI转换为抽象路径名来创建新的 File实例。

可以将一个已存在或不存在的文件或目录封装成file对象

常用函数

1、boolean createNewFile()

创建一个新的文件，不包含上级目录（使用时上级目录必须不存在，否则会引发异常）不存在时创建，存在时不创建

2、boolean mkdir()

创建此抽象路径名指定的目录。

3、boolean mkdirs()

创建此抽象路径名指定的目录，包括所有必需但不存在的父目录。

4、boolean exists()

测试此抽象路径名表示的文件或目录是否存在。存在true

6、String getName()

返回由此抽象路径名表示的文件或目录的名称（最后一级子目录）。（文件不存在也可用）

7、String getParent()

返回此抽象路径名父目录的路径名字符串；如果此路径名没有指定父目录，则返回 null。

8、File getParentFile()

返回此抽象路径名父目录的抽象路径名；如果此路径名没有指定父目录，则返回 null。

9、String getAbsolutePath()

返回此抽象路径名的绝对路径名字符串。

10、boolean delete()

删除此抽象路径名表示的文件或目录。（如果含有子文件或子文件夹则不能删除）（谨慎删除，不进回收站）（只能一级一级的删）（当文件被流正在操作，不能被删除，返回false）

11、boolean isFile()

测试此抽象路径名表示的文件是否是一个标准文件。（是的前提是存在）

12、boolean isHidden()

测试此抽象路径名指定的文件是否是一个隐藏文件。

13、String[] list()

返回一个字符串数组，这些字符串指定此抽象路径名表示的当前目录中的文件和目录。(得到子文件列表，若为空目录，返回一个空的数组)（必须应用于文件夹，否则产生空指针异常，若为系统及目录，也会发生异常）（包含隐藏文件）

14、File [] listFiles()

返回一个抽象路径名数组，这些路径名表示此抽象路径名表示的目录中的文件。(得到子文件列表)（必须应用于文件夹）

15、boolean renameTo(File dest)

重新命名此抽象路径名表示的文件。（可用来实现文件的移动）

16、isDirectory()

测试此抽象路径名表示的文件是否为目录。
